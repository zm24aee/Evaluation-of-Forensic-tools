Evaluation and Effectiveness of Volatility 3 and MemProcFS in Detecting Fileless Malware Through Memory Analysis

A controlled experimental research project comparing two mainstream memory forensic tools — Volatility 3 and MemProcFS — across six memory acquisition scenarios (three benign, three fileless) using automated Python pipelines and objective ground-truth evaluation
Project Overview
Fileless malware executes entirely in volatile memory without writing to disk, making conventional forensic methods ineffective. This project evaluates how well Volatility 3 and MemProcFS detect fileless malware artefacts, compares their detection accuracy, false positive rates, repeatability, artefact coverage, and analysis time — producing the first empirical cross-tool comparison of its kind.
